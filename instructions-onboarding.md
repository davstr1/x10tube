cette plage expliquerait qu'il y a trois manières d'utiliser cette extension. 

On recommande à l'utilisateur qu'il le pine.

L'autre manière c'est via un clic droit sur n'importe quelle page qui ouvre un menu contextuel.


Et la troisième manière c'est via le raccourci.

Je verrais ça en une page simple, verticale, avec un flow naturel :
En haut — un message de bienvenue court et direct. Genre "You're all set. Here's how to use StraightToYourAI." Pas de blabla, l'utilisateur vient d'installer, il veut savoir quoi faire.
Ensuite, les 3 méthodes présentées comme des cartes ou des blocs visuels, chacune avec un GIF ou une image annotée qui montre le geste exact. L'ordre compte :

Pin it (recommandé, mis en avant visuellement avec un badge "Recommended") — tu montres le puzzle icon → pin → le bouton qui apparaît dans la toolbar. C'est la méthode que tu veux qu'ils adoptent, donc elle est première et plus grande.
Right-click — tu montres un clic droit sur une page → le menu contextuel avec ton option. Simple screenshot annoté.
Keyboard shortcut — tu affiches le raccourci directement, avec une note que c'est customisable dans chrome://extensions/shortcuts.
